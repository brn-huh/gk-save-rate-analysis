# 🧤골키퍼(GK) 및 수비 전술 종합 평가지표 최종 명세서

본 명세서는 넥슨 `MatchDetail` API 데이터를 결합·가공하여 골키퍼의 **순수 방어력**, **빌드업 기여도**, **수비 라인 안정성**, **상대 유저의 전술 성향**까지 입체적으로 추출해 내기 위한 통합 스펙입니다.

---

## 1. 종합 방어력 및 영역별 선방 지표

### ① 종합 선방률 (Total Save %)

- **설명:** 상대가 날린 모든 유효 슈팅 중 골키퍼가 실점하지 않고 막아낸 비율입니다.
- **공식:** `(상대 effectiveShootTotal - 상대 goalTotal) / 상대 effectiveShootTotal * 100`

### ② 박스 안 선방률 (In-Penalty Save %)

- **설명:** 페널티 박스 안쪽에서의 혼전 상황이나 문전 슈팅에 대한 순발력 및 방어 능력입니다.
- **로직:** 상대 `shootDetail`에서 `inPenalty == true` 이면서 `result`가 `1(유효슛)` 또는 `3(골)`인 슛 중 `result == 1`인 비중.

### ③ 박스 밖 선방률 (Out-Penalty Save %)

- **설명:** 페널티 박스 바깥 중거리 슈팅에 대한 시야 확보, 다이빙 리치 및 위치 선정 능력입니다.
- **로직:** 상대 `shootDetail`에서 `inPenalty == false` 이면서 `result`가 `1(유효슛)` 또는 `3(골)`인 슛 중 `result == 1`인 비중.

## 2. 슈팅 직면 거리 기반 확장 통계

> 💡 **좌표 연산 기본 정의:** 골문 중앙 좌표를 $(x_g, y_g) = (1.0, 0.5)$로 가정할 때, 각 유효슛 좌표 $(x, y)$와의 유클리드 거리를 연산합니다.
> 
> 
> $$
> \text{Distance (m)} = \sqrt{(1.0 - x)^2 + (0.5 - y)^2} \times \text{경기장 실제 가로 길이 보정값}
> $$
> 

### ① 슈팅 직면 평균 거리 (Avg. Shot Distance)

- **설명:** 골키퍼가 마주한 모든 유효 슈팅의 평균 거리입니다. 수치가 짧을수록 위기 난이도가 높았음을 의미합니다.

### ② 거리 구간별 세부 지표 (히트맵/차트 연동용 4개 항목)

각 유효 슈팅의 계산된 거리($D$)에 따라 자동 분류하여 개별 `[유효슛 수 / 실점 수 / 선방률]`을 도출합니다.

- **초근거리 존 (`Shortest_Shot_Zone` / $0 \le D < 5\text{m}$):** 문전 난전, 탭인 슛 방어력 (순수 반사신경).
- **근거리 존 (`Close_Shot_Zone` / $5\text{m} \le D < 11\text{m}$):** 페널티 마크 부근, 완벽한 1대1 찬스 저지율.
- **중거리 존 (`Mid_Shot_Zone` / $11\text{m} \le D < 16.5\text{m}$):** 박스 내부 진입 슈팅 및 정교한 슛 메타 방어력.
- **원거리 존 (`Long_Shot_Zone` / $16.5\text{m} \le D$):** 박스 바깥 중거리 감아차기(Z+D) 및 파워샷 저지력.

### ③ 거리 가중치 선방 점수 (Distance-Weighted Save Score)

- **설명:** 슈팅 거리에 따른 난이도를 반영한 진짜 실력 점수입니다.
- **공식:** 선방(result=1)에 성공한 슛들의 `(1 / 슛 거리)` 누적 총합. (3m 앞 선방은 0.33점, 20m 밖 선방은 0.05점 적립하여 단순 횟수 통계 왜곡 방지)

### ④ 실점 평균 거리 (Avg. Conceded Shot Distance)

- **설명:** 내가 주로 어느 거리에서 골을 얻어맞고 무너졌는지 평균 패배 지점을 도출합니다.
- **공식:** `result == 3 (골)`인 슈팅들의 거리 값의 평균.

## 3. 인게임 슈팅 타입별 선방 지표 (`shootDetail` 기반)

상대 `shootDetail`에서 유효타(`result`가 1 또는 3)인 슈팅들을 `type` 코드로 필터링하여 각각의 선방률을 구합니다.

- **공통 공식:** `각 타입별 result가 1인 개수(선방) / 각 타입별 전체 유효슛 개수 * 100`

| 지표명 | 추출 조건 (`type` 코드) | 판별 및 세부 전술적 의미 |
| --- | --- | --- |
| **감아차기(Z+D) 방어율** | `type == 2` (finesse) | FC 온라인 핵심 메타인 궤적 슈팅에 대한 다이빙 방어력 |
| **낮은 슛(D+D) 방어율** | `type == 6` (low) | 키퍼 아래로 빠르게 깔려 들어오는 슈팅에 대한 반응 속도 |
| **파워샷 방어율** | `type == 12` (super) | 강력한 구속과 슛 파워를 이겨내고 저지하는 악력/저지력 |
| **헤더 선방률** | `type == 3` (header) | 세트피스 및 크로스 고공 분투 상황에서 역동작을 극복하는 반사신경 |
| **발 슈팅 선방률** | `type != 3` (header 제외) | 머리를 제외하고 상대가 발로 강하게 때린 슈팅에 대한 순수 방어율 |

## 4. 플레이어 수비 라인 및 상대 전술 평가 지표

### ① 수비 라인 붕괴율 (Defensive Line Collapse Rate %)

- **설명:** 내 수비진이 상대 공격수에게 뒷공간이나 문전 침투를 얼마나 허용했는지 평가하는 수비 안정성 지표입니다.
- **공식:** `(상대의 11m 이하 슈팅 횟수 / 상대의 전체 유효 슈팅 횟수) * 100`
- **의미:** 수치가 높을수록 골키퍼 탓이 아니라 수비 라인이 완전히 털려서 키퍼가 '독박 축구'를 했음을 증명합니다.

### ② 상대 슈팅 성향 유형 (Opponent Shot Depth Type)

- **설명:** 이 매치에서 상대 유저가 고집한 공격 패턴을 규명합니다.
- **분류 로직:** 경기 종료 후 상대의 `Avg. Shot Distance` 기준
    - **`~10m`**: **박스 파괴형 (Inside Box Slicer)** - 컷백이나 확실한 패스 연계로 골문 앞까지 썰어 들어오는 유저
    - **`10m ~ 16m`**: **하이브리드형 (Balanced Striker)** - 전술이 유연한 유저
    - **`16m 이상`**: **중거리 저격형 (Long-range Sniper)** - 박스 밖 Z+D나 게이지 꽉 채운 슛을 난사하는 유저

### ③ 상대 박스 진입 효율성 (Box Entry Efficiency)

- **설명:** 상대방이 내 수비 진영을 얼마나 효율적으로 깨고 들어와 질 좋은 슛을 쐈는지 나타내는 지표입니다.
- **공식:** `페널티 박스 안 슈팅 수 / 상대 전체 슈팅 수`

## 5. 상황별 위기 극복 및 경기 안정성 지표

### ① 1대1/오픈 위기 선방률 (Unassisted Shot Save %)

- **설명:** 어시스트 패스 없이 상대가 드리블로 수비 라인을 완전히 허물고 들어온 역습/독대 상황에서의 1대1 슈퍼세이브 능력입니다.
- **로직:** 상대 `shootDetail`에서 `inPenalty == true` 이고 `assist == false`인 유효슈팅 대비 선방률.

### ② 연계/컷백 방어율 (Assisted Shot Save %)

- **설명:** 상대의 패스 연계나 컷백에 의해 수비 밸런스가 무너진 타이밍에서 골키퍼가 역동작을 이겨내고 막아낸 비율입니다.
- **로직:** 상대 `shootDetail`에서 `assist == true`인 유효슈팅 대비 선방률.

### ③ 슈팅 대비 실점률 (Goals / SoTA)

- **설명:** 선방률의 정반대 개념으로, 상대의 유효 슈팅이 골문으로 날아왔을 때 골을 허용할 확률입니다. 수치가 0에 가까울수록 철벽임을 의미합니다.
- **공식:** `상대 goalTotal / 상대 effectiveShootTotal`

### ④ 90분당 실점률 (Goals Conceded per 90)

- **설명:** 경기 시간 기준, 1경기(90분)당 평균 몇 골을 허용하는지 나타내는 클래식 지표입니다.
- **공식:** `(상대 goalTotal / 총 경기 시간(분)) * 90`

## 6. 골키퍼 고유 빌드업 및 피지컬 지표 (`player` -> `status` 활용)

### ① 골키퍼 패스 성공률 (GK_PSR)

- **설명:** 골키퍼의 발밑 능력을 평가합니다. 패스 전개 안정성 및 롱킥의 정확도를 뜻합니다.
- **공식:** `GK 선수의 passSuccess / GK 선수의 passTry * 100`

### ② 골킥/롱패스 성향 지표 (Launch Rate %)

- **설명:** 골키퍼가 시도한 전체 패스 중 롱킥이 차지하는 비율입니다. 수치가 낮을수록 숏패스 빌드업 팀, 높을수록 다이렉트 롱볼 팀의 성향을 나타냅니다.
- **공식:** `(longPassTry + lobbedThroughPassTry + bouncingLobPassTry) / passTry * 100`

### ③ 공중볼 경합 성공률 (GK_AER)

- **설명:** 상대의 크로스나 코너킥 시 골키퍼가 직접 박스 안 공권을 장악(펀칭 및 캐칭)하여 슈팅 위기 자체를 사전 차단했는지 평가합니다.
- **공식:** `GK 선수의 aerialSuccess / GK 선수의 aerialTry * 100`

### ④ 시스템 부여 평점 (GK_Rating)

- **설명:** 스탯 숫자에 기록되지 않는 미세한 인게임 위치 선정, 슈퍼세이브 가산점 등이 포함된 엔진 자체의 최종 평가 점수입니다.
- **추출:** GK 선수의 `status` -> `spRating` 값.

## 7. 경기 운 및 외부 변수 제어 지표 (보정용)

### ① 골대 행운 횟수 (Frame Saved Count)

- **설명:** 골키퍼가 막은 게 아니라, 골대(포스트)를 맞아 실점 위기를 넘긴 유저의 '골대 운' 지표입니다.
- **추출:** 상대 `shootDetail` 배열에서 `hitPost == true`인 데이터의 총 개수.

### ② 불가항력 실점 수 (Unsavable Goals)

- **설명:** 골키퍼 능력 밖의 실점을 따로 집계하여 종합 선방률 하락에 대한 참작 사유를 제공합니다.
- **공식:** `상대 PK 실점 수 (type == 9) + 내 자책골 수 (my matchDetail -> shoot -> ownGoal)`
